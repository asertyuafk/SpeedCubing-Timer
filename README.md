# Speedcubing Timer

Веб-таймер для сборки кубика Рубика и других головоломок. Проект полностью разработан с помощью искусственного интеллекта (ИИ) и работает прямо в браузере.

## Попробовать онлайн

Вы можете запустить таймер без установки по ссылке: [Таймер](https://asertyuafk.github.io/SpeedCubing-Timer/)

## Основные возможности

* **Генерация скрамблов**: Поддержка кубов от 2x2 до 7x7, а также Pyraminx, Skewb, Square-1, Megaminx и Clock.
* **Подрежимы сборки**: Специальные режимы, включая сборку вслепую (Blind), одной рукой (OH), Mirror-куб, Void-куб и FMC.
* **Динамический фон**: Цвет страницы меняется в реальном времени, отражая близость к текущему рекорду.
* **Умная статистика**: Автоматический расчет метрик Best, Current, mo3, ao5, ao12 и ao100.
* **Система достижений**: 10 встроенных ачивок с уведомлениями и анимацией конфетти на Canvas.
* **Управление результатами**: Быстрое назначение штрафа +2 секунды, DNF или удаление сборки.
* **Импорт и экспорт**: Сохранение и загрузка истории сессий через файлы JSON.

## Технологический стек

* HTML5
* CSS3 (Адаптивный темный интерфейс, анимации)
* Ванильный JavaScript (Логика таймера, генерация скрамблов, расчеты)
* HTML5 Canvas (Эффекты конфетти)
* Web Storage API (`localStorage` для хранения истории и ачивок)

## Управление

* **Зажатие Пробела**: Подготовка таймера (индикатор меняет цвет на красный, затем на зеленый).
* **Отпускание Пробела**: Запуск отсчета времени.
* **Нажатие любой клавиши при работе**: Остановка таймера и автоматическая запись результата.
